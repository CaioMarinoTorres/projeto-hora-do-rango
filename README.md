> Projeto em desenvolvimento contínuo, criado para demonstrar **organização de código, versionamento e boas práticas no desenvolvimento backend com Java e Spring Boot**.
> A proposta é evoluir gradualmente, incorporando novas funcionalidades e melhorias ao longo do tempo.

---

## 🚀 Tecnologias

* Java 17+
* Spring Boot
* Spring Web
* Spring Data JPA
* Hibernate
* Gradle
* MapStruct
* Swagger (OpenAPI)
